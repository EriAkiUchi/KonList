# KonList  
Uma API Java capaz de pesquisar dados de animes e salvá-los em um banco de dados local. A idea surgiu da prática de criar listas de animes para assistir, semelhante ao MyAnimeList, mas extreamamente mais simples.   
Esta API foi desenvolvida com o intuito de praticar conceitos de Spring. Dentre elas:  
- Orientação a Objetos
- Separação de Responsabilidades
- Uso de JPA para conexão com banco de dados
- Operações com o banco 
- Injeção de dependência
- Criação de Entidade  
- Uso de Annotations
  
## Tecnologias utilizadas
- Java/Spring Boot
- JPA
- MYSQL/Docker
- Jikan API -> API com as informações dos animes

## Funcionalidades
- Pesquisa de dados sobre anime específico
- Salvar dados de anime no banco
- Remover anime do banco
